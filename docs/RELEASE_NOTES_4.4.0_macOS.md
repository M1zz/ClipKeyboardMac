ClipKeyboard for Mac v4.4.0

한국어

맥에서 순서를 직접 정하고, 그 순서가 아이폰까지 이어집니다 — 세 기기가 한 몸처럼 움직여요.

- 맥에서 단축어 순서를 끌어다 바꿔요, 맥 앱 목록에서 단축어를 드래그해 원하는 순서로 정할 수 있습니다 — 여기서 정한 순서는 아이폰과 키보드까지 그대로 이어지고, 아이폰에서 바꾼 순서도 맥의 메뉴바 검색·플로팅 패널·메인 창에 똑같이 나타나요
- 카테고리 관리가 한 화면으로 정리됐어요, 흩어져 있던 카테고리 관리 진입점을 하나로 통일해서 어디서 들어와도 같은 화면에서 켜고 끄고 정리할 수 있습니다
- 더 조용하고 사적인 앱이 됐어요, 사용 통계를 수집하던 분석 도구를 앱에서 완전히 걷어냈습니다 — 이제 맥 앱은 외부로 아무 통계도 보내지 않아요

English

Set the order yourself on Mac, and it carries all the way to iPhone — three devices moving as one.

- Drag to reorder snippets on Mac, drag snippets into any order in the Mac app list — the order you set carries over to your iPhone and the keyboard, and an order you set on iPhone shows up the same in the Mac menu-bar search, floating panel, and main window
- Category management, now one screen, the scattered entry points for managing categories are unified into a single screen, so you toggle and tidy categories in the same place no matter how you get there
- A quieter, more private app, we removed the analytics tool that collected usage stats entirely — the Mac app now sends no analytics anywhere

---

배포용 요약 (한국어)

• 맥에서 단축어를 끌어다 순서를 바꾸고, 그 순서를 아이폰·키보드와 양방향 동기화
• 카테고리 관리 진입점을 단일 화면으로 통일
• 사용 통계 분석 도구 완전 제거 — 외부 전송 없음

Release Summary (English)

• Drag to reorder snippets on Mac, with two-way sync to iPhone and the keyboard
• Category management unified into a single screen
• Analytics tool removed entirely — nothing sent externally
